## Ejecutar Aplicativo

**tramites-servicios-spa** podrá ejecutarse de las siguientes dos manera:

### Docker

    docker build --tag=tramites-servicios-spa . --no-cache

### Docker Compose

    docker-compose up


Ingrese a la siguiente URL para visualizar el aplicativo  http://localhost:8185/
